# ASCENSION - Juan Grey Private Mentorship

Landing page for Juan Grey's 90-day private mentorship program.

## Overview

This is a standalone HTML landing page designed for a premium mentorship program focused on structured performance, discipline, and personal transformation.

## Features

- **Self-contained HTML** - No build process required
- **Tailwind CSS** via CDN
- **Responsive design** - Mobile-first approach
- **Scroll animations** - Smooth fade-in effects
- **Application form** - Client-side validation
- **Dark cinematic theme** - Gold accents on dark background

## Deployment

The site is configured for deployment on Vercel and serves the `index.html` file directly.

### Files

- `index.html` - Main landing page
- `vercel.json` - Vercel deployment configuration
- `attached_assets/` - Images and media files

## Local Development

Simply open `index.html` in your browser. No server or build tools required.
